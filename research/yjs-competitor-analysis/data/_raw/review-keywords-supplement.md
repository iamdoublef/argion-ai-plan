## 采集说明

### 方法
1. 使用 Playwright browser 访问每个产品的 Amazon 产品页 (`https://www.amazon.com/dp/{ASIN}`)
2. 通过 JavaScript 提取页面中 "Customers say" AI生成摘要区域
3. 提取评价维度标签（如 Functionality, Quality, Ease of use 等）及其提及次数
4. 根据摘要文本判断正面/负面/混合情感倾向

### 局限性
- Amazon "Customers say" 是 AI 生成的摘要，可能不完全反映所有评价
- 维度标签的"提及次数"是 Amazon 统计的，非精确数字（显示为 1.2K 等）
- Vesta Precision 评价数极少(67条)，数据代表性有限
- Wevac 是耗材产品，评价维度与封口机不同
- 未能获取每个维度的详细正面/负面拆分（需要点击每个标签进入子页面）
