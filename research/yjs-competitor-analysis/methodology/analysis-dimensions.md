## 分析框架：三层递进

```
Layer 1: 竞品特性矩阵 (What)    → 产品层面差距

Layer 3: 数字化竞争力 (Now What) → 数字化行动方向
```

## Layer 1：竞品特性矩阵

### A. 产品基础信息
| 维度 | 说明 | 数据来源 |
|------|------|---------|
| 品牌与型号 | 品牌名、产品型号 | 官网/Amazon |
| 产品类别 | 外抽式/腔式/手持式/低温烹饪/配件 | 官网 |
| 目标市场 | 家用/商用/专业级 | 产品定位 |
| 上市时间 | 首次上市年份 | Amazon/官网 |

### B. 技术规格
| 维度 | 说明 | 数据来源 |
|------|------|---------|
| 密封方式 | 外抽式(External)/腔式(Chamber)/手持式(Handheld) | 产品页 |
| 真空强度 | kPa / inHg | 规格表 |
| 密封宽度 | mm，单条/双条 | 规格表 |
| 功率 | W | 规格表 |
| 泵类型 | 油泵/干泵/活塞泵 | 规格表 |
| 尺寸 | L×W×H (mm/inch) | 规格表 |
| 重量 | kg/lbs | 规格表 |

### C. 功能特性
| 维度 | 评分方式 | 说明 |
|------|---------|------|
| 干湿模式 | 有/无 | 是否支持液体食品 |
| 脉冲真空 | 有/无 | 精细控制真空度 |
| 罐装/瓶装密封 | 有/无 | 配件兼容性 |
| 内置切刀 | 有/无 | 便利性 |
| 卷袋存储 | 有/无 | 集成度 |
| 自动袋口检测 | 有/无 | 智能化 |
| 预设程序 | 数量 | 智能模式数 |
| 连接能力 | Wi-Fi/蓝牙/无 | IoT能力 |
| App控制 | 有/无 | 智能生态 |

### D. 价格与价值
| 维度 | 说明 | 数据来源 |
|------|------|---------|
| 零售价 | USD (Amazon) | Amazon产品页 |
| 耗材价格 | 袋/卷单价 | Amazon |
| 性价比指数 | 功能数/价格 | 计算 |
| 促销频率 | 折扣力度和频率 | 价格追踪 |

### E. 用户评价
| 维度 | 说明 | 数据来源 |
|------|------|---------|
| Amazon评分 | 1-5星 | Amazon |
| 评价数量 | 总评价数 | Amazon |
| 好评关键词 | Top 5正面关键词 | 评价分析 |
| 差评关键词 | Top 5负面关键词 | 评价分析 |
| 退货率指标 | 如有 | Amazon |

### F. 售后与认证
| 维度 | 说明 | 数据来源 |
|------|------|---------|
| 保修期限 | 年 | 官网 |
| 认证 | ETL/CE/FDA/LFGB等 | 官网/产品页 |
| 客服渠道 | 电话/邮件/在线 | 官网 |

## Layer 2：价值曲线分析

### 竞争要素（Strategy Canvas X轴）

12个关键竞争要素，每个按1-10评分：

| # | 竞争要素 | 评分依据 |
|---|---------|---------|
| 1 | 价格竞争力 | 同类产品价格排名（越低越高分） |
| 2 | 密封性能 | 真空强度 + 密封宽度 + 稳定性 |
| 3 | 产品耐用性 | 用户评价中耐用性相关反馈 |
| 4 | 功能丰富度 | Layer 1 C类功能数量加权 |
| 5 | 易用性 | 用户评价中易用性相关反馈 |
| 6 | 工业设计 | 外观、材质、做工评价 |
| 7 | 智能化程度 | IoT/App/智能模式综合评分 |
| 8 | 品牌知名度 | 搜索量、评价数、市场份额 |
| 9 | 渠道覆盖 | 电商平台数 + 线下渠道 |
| 10 | 耗材生态 | 袋/卷种类、价格、兼容性 |
| 11 | 售后服务 | 保修期 + 客服响应 + 用户反馈 |
| 12 | 产品线完整度 | 产品类别数 × 型号数 |

### 四步动作框架输出
- **消除（Eliminate）**：哪些行业惯例可以去掉
- **减少（Reduce）**：哪些要素可以降低投入
- **提升（Raise）**：哪些要素需要超越行业标准
- **创造（Create）**：哪些全新要素可以引入

## Layer 3：数字化竞争力评估

### A. 智能产品能力（权重25%）
| 维度 | 评分标准 |
|------|---------|
| IoT连接 | 0=无, 5=蓝牙, 10=Wi-Fi+App |
| App功能 | 0=无App, 5=基础控制, 10=AI推荐+社区 |
| 智能烹饪推荐 | 0=无, 5=预设, 10=AI自适应 |
| OTA更新 | 0=无, 10=支持 |

### B. 电商与数字营销（权重30%）
| 维度 | 评分标准 |
|------|---------|
| 电商平台覆盖 | 平台数量和排名 |
| SEO/搜索可见度 | 关键词排名 |
| 社交媒体影响力 | 粉丝数 + 互动率 |
| 内容营销质量 | 视频/食谱/教程数量和质量 |
| KOL合作 | 合作数量和影响力 |

### C. 数据与AI应用（权重20%）
| 维度 | 评分标准 |
|------|---------|
| 用户数据采集 | 0=无, 5=基础, 10=全链路 |
| 个性化推荐 | 0=无, 10=AI驱动 |
| 智能客服 | 0=无, 5=FAQ, 10=AI对话 |
| 供应链AI | 0=无, 10=AI优化 |

### D. 数字化客户体验（权重25%）
| 维度 | 评分标准 |
|------|---------|
| 官网体验 | 速度/设计/功能综合评分 |
| 移动端体验 | 响应式/App质量 |
| 在线购买流程 | 便捷度 |
| 数字化售后 | 在线报修/追踪/知识库 |

## 竞品选择标准

### 必选竞品
- **FoodSaver**（Newell Brands）：北美市场领导者，家用真空封口机代名词
- **Anova**（Electrolux）：低温烹饪领域领导者，智能化标杆

### 选择标准（Top 5中其余3个）
1. 与亚俊氏产品线有直接竞争关系
2. 在Amazon真空封口机品类有显著市场份额
3. 覆盖不同价格带和定位（高端/中端/性价比）
4. 优先选择有自有品牌的制造商（与亚俊氏模式相似）

### 候选品牌池
- Nesco / GERYON / Mueller / Anova / Seal-a-Meal / Avid Armor / VacMaster / Crenova
