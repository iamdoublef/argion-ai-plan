# Prompt模板库设计方案

> 项目：yjs-content-pilot
> 日期：2026-02-14

---

## 设计原则

1. **模块化**：System Prompt（品牌调性）与 Task Prompt（具体任务）分离，便于复用
2. **变量化**：所有产品相关信息通过变量注入，同一模板适用不同SKU
3. **可检验**：每个模板附带质量检查清单，确保输出一致性
4. **可迭代**：模板版本化管理，根据使用反馈持续优化

## 模板架构

### 层级结构

```
[品牌调性 System Prompt]  ← 所有模板共享
    ↓
[产品线上下文]            ← 按产品线切换
    ↓
[内容类型 Task Prompt]    ← 具体任务指令
    ↓
[产品变量输入]            ← 每个SKU不同
    ↓
[输出格式约束]            ← 字数、结构、关键词
```

### 品牌调性 System Prompt（共享）

```
你是亚俊氏（Argion）旗下品牌 {brand_name} 的资深电商文案专家。

品牌定位：
- Wevac：专业级真空保鲜，面向注重食品品质的家庭用户和美食爱好者
- Vesta：高性价比真空保鲜入门选择

品牌语气：
- 专业但不冷冰冰，像一个懂行的朋友在推荐好产品
- 强调实际使用场景和真实好处，避免空洞的营销话术
- 数据驱动：用具体数字说话（保鲜时长、节省金额等）

核心价值主张：
- 创新技术（Liquid Mode、Smart Seal、Smart Vac）
- 专业品质（17年真空技术经验，全球认证）
- 环保承诺（可降解真空袋，OK Compost + BPI双认证）

禁用词/表达：
- 不使用"最好的"、"第一"等绝对化用语
- 不贬低竞品
- 不做未经验证的健康声明
```

### 输入变量模板

每个SKU需要准备以下变量：

```yaml
product_name: "Wevac CV10 Chamber Vacuum Sealer"
brand: "Wevac"
category: "真空封口机"
subcategory: "Chamber Vacuum Sealer"
key_specs:
  - "双泵系统，极限真空度 -99.9kPa"
  - "10英寸封口条"
  - "不锈钢机身"
  - "LCD触控面板"
key_benefits:
  - "液体食品也能完美真空封装（汤、酱汁、腌料）"
  - "商用级真空度，保鲜效果提升5倍"
  - "一键操作，无需学习曲线"
target_audience: "美食爱好者、Sous Vide玩家、批量备餐用户"
price_range: "$299-$399"
target_keywords:
  - "chamber vacuum sealer"
  - "liquid vacuum sealer"
  - "sous vide vacuum sealer"
  - "commercial vacuum sealer for home"
competitors: ["FoodSaver", "Anova", "VacMaster"]
unique_selling_points:
  - "唯一同价位带双泵系统的家用腔式封口机"
  - "Liquid Mode专利技术"
```

## 各模板文件说明

| 模板文件 | 内容 | 输出格式 |
|----------|------|----------|
| `product-description.md` | Amazon Listing（标题+五点+描述） | 标题≤200字符，五点各≤500字符，描述≤2000字符 |
| `aplus-page.md` | A+页面（品牌故事+模块内容） | 5-7个模块，每模块标题+正文+图片建议 |
| `ad-copy.md` | PPC广告+社媒帖子 | SP/SB广告标题+描述，Instagram/Facebook帖子 |
| `translation-guide.md` | 多语言翻译+本地化 | 保持品牌调性的目标语言内容 |
| `image-generation.md` | 产品图片生成Prompt | Midjourney/Ideogram可直接使用的Prompt |

## 质量检查清单（通用）

每次AI内容生成后，人工审核需检查：

### 事实准确性
- [ ] 产品规格参数正确
- [ ] 功能描述与实际一致
- [ ] 价格/促销信息准确
- [ ] 认证信息（ETL/CE/FDA等）正确

### 品牌一致性
- [ ] 语气符合品牌调性
- [ ] 未使用禁用词/表达
- [ ] 品牌名称拼写正确
- [ ] 核心价值主张体现

### SEO优化
- [ ] 目标关键词自然融入
- [ ] 标题包含主要关键词
- [ ] 五点描述覆盖长尾关键词
- [ ] 无关键词堆砌

### 平台合规
- [ ] 符合Amazon内容政策
- [ ] 无夸大/虚假宣传
- [ ] 无竞品品牌名提及（除对比图表）
- [ ] HTML标签使用正确（A+页面）
