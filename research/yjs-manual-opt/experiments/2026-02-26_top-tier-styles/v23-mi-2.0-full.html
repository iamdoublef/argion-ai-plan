02安全须知

为确保使用安全，在使用本电器产品前，请阅读并遵守以下安全预防措施：

警告 WARNING
• 本机仅供家庭使用，请勿用于非设计用途。
• 请勿让儿童玩耍本机或电源线。避免窒息危险。
• 严禁在有易燃易爆气体的环境中使用。
• 密封完成后，密封区域温度较高，请勿触摸，以免烫伤。
• 请勿将主机或电源线浸入水中或其他液体中。
注意 CAUTION
• 长时间不使用时，请勿扣上锁扣，以免密封圈受压变形。
• 请妥善保管本手册。
产品及真空袋使用提示

• 本机只适用于带有空气通道纹路的真空袋。
• 装袋时，请务必在袋口和食物之间预留至少 5cm (2 inch) 的空间。
• 如果需要封装质地柔软或含水分多的食物，建议先冷冻后再进行真空封装。

04产品功能
控制按键

Vac&Seal / Start 抽气密封
执行选定的真空及密封模式。

Stop 停止
随时终止当前进程。

Vac Mode 真空模式选择
Normal (标准) / Gentle (轻柔) / Liquid (液体)。

Pulse Vac 手动真空
长按控制抽气，松开暂停。精准控制易碎食物的真空度。

Seal 密封
立即执行密封动作。

Accessories 附件类
Mason Jar (梅森罐) / Canister (真空罐) / Marinate (腌制)。

Extend Seal Time 延长密封时间
应对厚袋子或潮湿袋口，增强密封强度。

06操作指引
如何使用切袋刀
1
打开面盖及切袋刀架。
2
拉出所需长度的卷袋。
3
将刀架压在袋子上，滑动刀头进行整齐切割。
制作袋子并密封
1
将切好的袋子一端平整放入真空腔。
2
关闭面盖并下压把手锁紧。
3
点击 Seal，待指示灯熄灭后解锁。
V23 使用说明书7
07如何使用真空密封
1
将袋口平整放入真空腔。
2
根据食材选择 Normal 或 Gentle。
3
点击 Vac&Seal / Start 开始工作。
密封液体建议

保持袋子直立，袋内预留至少 18cm 空间，模式选择为 Liquid。

手动真空 (Pulse Vac) 技巧

长按 Pulse Vac，观察抽气进度；松开即停止抽气并保压；达到理想状态后点 Seal 封口。

V23 使用说明书8
08故障排除
故障现象可能原因及解决方案
不能开机 • 检查电源线是否插紧。
• 检查电源线是否破损。
抽气不干净 • 确保袋口完全放入真空腔内。
• 检查密封圈是否变形或破损。
• 确保使用带有纹路的真空袋。
密封不理想 • 检查密封区域是否有水分、颗粒。
• 尝试开启 Extend Seal Time。
机器漏气报警 • 检查面盖把手是否完全压下锁紧。
• 检查袋子是否被锋利物品刺穿。
V23 使用说明书9
09维护保养

• 清洁前请务必拔下电源，并等待发热器冷却。
• 每周使用软毛刷清洁密封圈、真空腔及液体槽。
• 液体槽支持取出清洗，但不可放入洗碗机。
