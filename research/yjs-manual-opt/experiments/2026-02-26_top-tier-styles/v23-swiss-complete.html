CH.01 — SAFETY

01安全须知

警告 WARNING
  • 本机仅供家庭使用，请勿用于其他非设计用途。
  • 请勿让儿童玩耍本机或电源线。儿童、行动不便或认知受限人士应在成年人监督下使用。使用期间请留意儿童，避免意外发生。
  • 小零件、真空袋可能造成窒息危险，请勿让儿童接触。
  • 请勿在高湿高温等环境使用本机。严禁在有易燃易爆气体的环境中使用。
  • 请确保电源电压与机器铭牌标示一致。
  • 机器工作时请勿离开。
  • 密封完成后，密封区域温度较高，请勿触摸，以免烫伤。
  • 使用后请切断电源。
  • 从电源插座上拔出插头时请手握插头拔出，请勿拉扯电源线。
  • 请勿将主机或电源线浸入水中或其他液体中。
  • 电源线需要直接插接符合安全规范的电源插座上。
  • 如发现插头、电源线或主机损坏，请立即停止使用并联系制造商或授权维修点处理，请勿自行拆修。
  • 电源线请勿悬挂于台面边缘，以免儿童拉扯或意外绊倒。
注意 CAUTION
  • 请妥善保管本手册。
  • 本机无需添加任何润滑剂，请勿使用有机溶剂清洁。
  • 长时间不使用时，请勿扣上锁扣，以免密封圈受压变形。
  • 请勿将本产品与生活垃圾一同丢弃，请根据您所在地区关于电子废弃物的法规进行回收或处置。
提示 NOTICE
  • 本产品适用于家庭食品保鲜用途。
  • 首次使用前，请完整阅读本说明书。
  • 请保留购买凭证，以便享受保修服务。

CH.02 — USAGE TIPS

02产品及真空袋使用提示

  • 本机只适用于带有空气通道纹路的真空袋。
  • 对于带有尖点或棱角的物品，需用防刺穿片、纸巾等柔软包材包裹棱角，以防刺破真空袋。
  • 真空袋可用于加热或微波炉加热，但必须先在袋角剪开一个小口透气。
  • 用于储存的真空袋可重复使用，但用于加热或微波加热时仅可使用一次。储存过海鲜或油腻食品的真空袋不建议重复使用。
  • 放置袋子时，袋口必须平整地跨过发热器。
  • 装袋时，请务必在袋口和食物之间预留至少 2 inch（5 cm）的空间。
  • 装入液体、酱汁或带液体的食物时，请避免将真空袋袋口弄湿。建议装入后将袋口擦干，以免影响密封效果。
  • 如果需要质地柔软、含水分多的食物在包装后保持形状，可选择冷冻后进行真空封装。
  • 勿在面盖上放置重物，有可能损坏机器。

CH.06 — OPERATION

06操作指引

6.1 如何使用切袋刀
  1. 1打开面盖。
  2. 2打开切袋刀架。
  3. 3从卷袋上拉扯出所需长度的袋子。
  4. 4将切袋刀移至最左或最右边，再使刀架下压着袋子。
  5. 5将切袋刀下压滑动划过袋子，以进行整齐的切割。
6.2 如何使用密封功能

从卷袋中制作袋子并密封：

  1. 1按以上如何使用切袋刀制作出一个两端开口的袋子。
  2. 2将袋子的一端开口平整放入真空腔内。
  3. 3关闭面盖，并将把手向下压以锁紧面盖。
  4. 4点击 Seal 按键，并等待机器完成工作。

单独密封袋子：

  1. 1将袋子放入需要封装的物品，并将袋口放入真空腔。
  2. 2关闭面盖，并将把手下压以锁紧面盖。
  3. 3点击 Seal 按键，并等待机器完成工作。
6.3 如何延长密封时间
若需延长密封时间，需在操作前点击 Extend Seal Time 按键，使该按键指示灯点亮。延长时间设置将持续生效。若要恢复默认密封时间，请在下一次操作前再次点击该按键。当机器进入休眠或断电重新开机，密封时间将会恢复默认设置。
注意
启用 Extend Seal Time 功能时，将暂时禁用智能密封调节功能，连续工作时每次工作间隔需大于 20s，并打开面盖以便机器散热。

CH.06 — OPERATION (CONT.)

06操作指引（续）

6.4 如何使用真空密封功能

真空密封固体物品：

  1. 1将袋口放入真空腔。
  2. 2关闭面盖，并下压把手以锁紧面盖。
  3. 3点击 Vac Mode 按键选择模式：Normal 或 Gentle。
  4. 4点击 Vac&Seal / Start 按键启动工作。
  5. 5若需终止工作，请点击 Stop 按键。
在采用 Gentle 模式封装贵重物品前，建议先采用其它物品尝试真空后的压缩程度是否满足需求，以免压碎物品。

真空密封液体：

  1. 1保持袋子直立，确保液体液面始终低于发热器，液体与袋口之间建议留出 7 inch（18 cm）的空间。
  2. 2将袋口放入真空腔。
  3. 3关闭面盖，并下压把手以锁紧面盖。
  4. 4点击 Vac Mode 按键选择 Liquid 模式。
  5. 5点击 Vac&Seal / Start 按键启动工作。
6.5 如何使用手动真空

方法一（在自动真空过程中切换至 Pulse Vac 模式）：

  1. 1长按 Pulse Vac 按键进行真空抽气。
  2. 2松开按键则暂停抽气。
  3. 3达到所需真空状态后，点击 Seal 按键进行密封。

方法二（直接使用 Pulse Vac 模式）：

  1. 1将袋口放入真空腔。
  2. 2关闭面盖，并下压把手以锁紧面盖。
  3. 3长按 Pulse Vac 按键进行真空抽气。
  4. 4松开按键则暂停抽气。
  5. 5达到所需真空状态后，点击 Seal 按键进行密封。
在手动真空过程中，长按按键抽气，松开暂停，此时会处于保压的状态。若 10 秒内不进行再次操作，机器将自动释放压力，并退出手动抽气模式。

CH.06 — ACCESSORIES

06操作指引（附件使用）

6.6 真空软管用法
  • 对梅森罐适配器和带阀门拉链袋抽真空时，圆形吸嘴采用方向①。
  • 对真空罐和瓶塞抽真空时，根据抽气口大小选择方向①或②。
  • 真空软管扁形吸嘴③用于连接机器面盖上的外抽气口③。
6.7 如何抽梅森罐
  1. 1闭合机器面盖。
  2. 2将罐盖放到梅森罐罐口，不需拧上螺纹盖（如有）。
  3. 3将适配器安装到梅森罐上。
  4. 4将真空软管扁形吸嘴插入机器面盖上的外抽气孔。
  5. 5压下把手以锁紧面盖。
  6. 6点击 Accessories 按键，选择 Mason Jar。
  7. 7将真空软管圆形吸嘴覆盖适配器的抽气孔，并用手轻微按压住。
  8. 8点击 Vac&Seal / Start 按键启动工作。
  9. 9工作结束，取下真空软管及适配器。
6.8 如何抽真空罐
  1. 1操作前，请确认真空罐与本机真空软管兼容。
  2. 2确认真空罐处于可抽真空（Vacuum）的状态。
  3. 3将真空软管扁形吸嘴插入机器面盖上的外抽气孔。
  4. 4压下把手以锁紧面盖。
  5. 5点击 Accessories 按键，选择 Canister。
  6. 6将真空软管圆形吸嘴覆盖真空罐抽气阀的抽气孔，并用手轻微压住。
  7. 7点击 Vac&Seal / Start 按键启动工作。
  8. 8工作结束，取下真空软管。

6.9 如何腌制
  1. 1将食物覆盖酱料，并放到真空罐中。

  7. 7点击 Vac&Seal / Start 启动工作。
  8. 8工作结束，取下真空软管。

CH.07 — LIQUID RECOVERY

07故障与排除（续）

如因特殊原因，导致液体进入机器内部的处理方案

液体进入真空腔中：

  1. 1立即关闭电源，等待至少 5 分钟，让机器发热器冷却。
  2. 2将液体槽取出，并清理擦干液体。
  3. 3将真空腔中的液体擦干。
  4. 4将机器底部铺垫厨房纸或托盘，重新通电。
  5. 5合上面盖，长按 Pulse Vac 按键抽气 20 秒，然后点击 Stop 排气，重复 5 次，让内部管道的液体排出。
  6. 6打开面盖，在机器底部铺垫厨房纸或托盘，将机器放置于通风处晾干 48 小时，期间不要翻倒机器。

液体进入发热器：

  1. 1请勿翻倒机器，关闭电源，等待 10 分钟，让发热器冷却及内部液体排出。
  2. 2清理和擦干机器接水盘、真空腔、底部的液体。
  3. 3打开面盖，在机器底部铺垫厨房纸或托盘，将机器放置于通风处晾干 24 小时，期间不要翻倒机器。
  4. 4将机器底部铺垫厨房纸或托盘，重新通电。
  5. 5合上面盖，长按 Pulse Vac 按键抽气 20 秒，然后点击 Stop 排气，重复 5 次，让内部管道的液体排出。
  6. 6打开面盖，在机器底部铺垫厨房纸或托盘，将机器放置于通风处晾干 48 小时，期间不要翻倒机器。
注意
如果长期不清洁，内部管道可能会堵塞，影响抽气性能，此时请联系售后进行管道更换。

CH.08 — MAINTENANCE

08维护保养

  • 确保使用机器的工作台平整、干净、干燥。
  • 在不使用机器时请不要将机器锁扣锁上，以防密封圈被长时间挤压，影响功能。
  • 需每周一次对机器的密封圈、密封压条、发热器、液体槽、真空腔进行清洁。
  • 发热器和密封压条在工作后会残留高温，请待机器冷却后再进行保养。
  • 清洁前拔下机器电源。
  • 请采用软毛刷清洁密封圈、密封压条、发热器、液体槽、真空腔、真空软管。避免采用硬毛刷，以免损坏机器。
  • 每天工作结束后，取出液体槽清洁擦干，并用湿毛巾或纸巾清理真空腔内残留的碎屑、灰尘。
  • 每次工作结束后，检查密封圈是否已经变形严重，高温胶布是否已经破损/脱落，如有，则请及时替换。
  • 液体槽和真空软管不支持高温清洁，如洗碗机清洁。

CH.09 — VACUUM PROPERTIES

09真空包装特性

真空包装是通过去除密封容器中的大部分空气来延长食品的寿命，并防止空气回流到密封容器中，从而减少氧化。真空包装有助于保持风味和整体质量。它还有助于抑制好氧微生物的生长，在某些条件下可能的现象有：

  • 霉菌 — 霉菌在低氧环境下无法繁殖，真空包装能够抑制霉菌的繁殖。
  • 酵母菌 — 酵母需要水、糖和中等温度才能繁殖，可以在有空气或无空气的情况下生存。减缓酵母的生长需要冷藏，冷冻则可以有效抑制。
  • 细菌 — 导致难闻的气味、变色或变柔软。在适当的条件下，肉毒梭菌可以在没有空气的情况下生长，且无法通过气味或味道检测到。

与任何其他储存方式一样，在食用前检查食品是否变质是很重要的。在 -4°C 或更低温度下，可以显著减少微生物的生长。对于长期储存，建议进行真空封装后保持冷冻。

NOTICE
真空包装不能代替冷藏或冷冻。任何需要冷藏的易腐食品在真空包装后仍必须保持冷藏或冷冻。
存储指南
  • 蔬菜在真空包装前需要先焯水。将蔬菜放入沸水中煮熟，然后浸入冷水中，最后用毛巾吸干水分再进行真空包装。
  • 十字花科蔬菜（花椰菜、芽菜、卷心菜等）在储存过程中会自然排放气体，煮熟、真空封装后建议存放在冰箱里。
  • 切忌真空储存大蒜或蘑菇等真菌食品。抽真空后会产生有害物质，食用会影响健康。
